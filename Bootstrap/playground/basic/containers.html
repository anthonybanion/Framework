Container inside a container

You can put a container inside another container in Bootstrap, but it's not ideal for flexible layouts.
Let me explain:

  1. A container in Bootstrap is designed to center and automatically set margins for the main content.
  2. If you put one container inside another, the margins and width are reset, and you can run into strange alignment issues.
  3. For internal layouts (for example, rows and columns, or flexbox), the correct way to do this is to use:
    • row and col- for grids.
    • d-flex and flexbox utilities for aligning elements.
